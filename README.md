# spiralworks.ai

built using pure.css, three.js, vite, node, and gh-pages

## development

test locally:
```shell
$ npx vite
```

## deploying

build:
```shell
$ npm run build
```

push to github:
```shell
$ git add .
$ git commit -m "commit msg"
$ git push -u origin main
```

deploy:
```shell
$ npm run deploy
```
